# what is chiara64 ?

Chiara64 is  a  most advanced  CISC computer architecture in the world. 

Chiara64 allows to create incredible applications on a flexible and simple computer architecture.



[more infos](https://thechiara.com)


# what is chiara64 emulator ?


Is a c written program. And it is the official emulator of chiara64 binary code. 



Is under development (alpha), some features may not  work. 




# how to use it ? 


First you need to compile it. 

Is very simple, just clone this repo and do : 


`gmake` (chiara64 emulator needs a gnu syntax make software) 


After run the makefile, chiara64 emulator will be compiled. 

So you can run it with : 


`./chia64-emul [YOUR BIN FILE]`

You can install it to your GNU/linux / BSD system : 

`gmake install`
